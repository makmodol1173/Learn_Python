**рззрзи. Descriptor Protocol ржХрзА?**

Descriptor рж╣рж▓рзЛ ржПржоржи object ржпрзЗржЯрж╛рж░ ржоржзрзНржпрзЗ special method ржерж╛ржХрзЗ:

>- Descriptor Protocol тЖТ `__get__`, `__set__`, `__delete__` method define ржХрж░рж╛ред
